# Zogra Render Engine

A simple WebGL2 renderer.


## Build Requirement
- Node @latest
- Typescript @3.7+

## Supported Platforms
- Chrome 79 (Desktop)
- FireFox 72 (Desktop)
- Chrome 79 (Android)
- ...

## Screen Shots

![Screen shot](./docs/assets/img/screen-shot-0.png)

## Features
- Basic 3D mesh rendering.
- Line rendering.
- Shader & material managment.
- Image texture & render texture.
- Basic scene manager.
- Input system.
